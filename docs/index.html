Generate full‑stack apps from a single source.

Locus is a source‑to‑source compiler that turns .locus files into a production‑ready Next.js frontend and an Express API. Typed entities, declarative UI, and design tokens—compiled.

Get started npm i locus
TypeScriptFirst‑class toolchain
Next.jsApp router scaffolding
ExpressCRUD routes and validation
Design SystemCSS variables generated
locus example 
database {
  entity Post {
    title: String @unique
    published: Boolean @default(false)
  }
}

page Home {
  state { greeting: String = "Hello, Locus!" }
  ui { <div>{greeting}</div> }
}
CLI 
# Build from .locus files to /generated
locus build --src . --out generated

# Dev mode with watch
locus dev --src .

# New project
locus new my-app

Why Locus?

🧠

Declarative and typed

Entities, UI, and actions are strongly‑typed. The compiler validates and generates predictable code.

⚡

Fast toolchain

Chevrotain parser + Jest tests + incremental builds. Designed for speed and iteration.

🧩

Full‑stack output

Next.js pages/components, Express routes, Prisma schema, and theme CSS tokens—all generated.

🎛️

Design system aware

Colors, spacing, radii, and typography become CSS variables your app can consume directly.

🧪

TDD‑friendly

Generators return strings—easy to snapshot test for stable outputs and safe refactors.

🚀

CLI that scales

Build, dev, new, db, deploy—scriptable commands that fit CI and local workflows.

Install

npm
npm i -g locus
# or project‑local
npm i -D locus

Quickstart

commands
# From your project root with .locus files
locus build --src . --out generated
